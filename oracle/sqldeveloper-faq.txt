Q: SOCKS proxy (-DsocksProxyHost=127.0.0.1 in sqldeveloper.conf has no effect)
A:
  The JVM arg socksProxyHost will be ignored by the driver. Use connection property oracle.net.socksProxyHost
  https://docs.oracle.com/en/database/oracle/oracle-database/21/jajdb/oracle/jdbc/OracleConnection.html#CONNECTION_PROPERTY_SOCKS_PROXY_HOST

  Connection properties, "Advanced"
  oracle.net.socksProxyHost 127.0.0.1
  oracle.net.socksProxyPort 1080
  oracle.net.socksRemoteDNS true

Q: import csv
A:
  don't try to import CSV with Oracle SQL Developer 20.x or below:
  it treats line breaks in values as record separators despite
  the quoting.

Q: debug stored procedure: ORA-24247: network access denied by access control list (ACL)
A: 
Oracle is disallowed to make a TCP connection to client's host
workaround: use DBMS_DEBUG instead of DBMS_DEBUG_JDWP in debugger settings (supported by SQL Developer 20+)
fix: 

    BEGIN
     DBMS_NETWORK_ACL_ADMIN.APPEND_HOST_ACE
     (
     host => 'subdomain.acme.com',
     lower_port => 10000,
     upper_port => 65535,
     ace => xs$ace_type(privilege_list => xs$name_list('jdwp'),
     principal_name => 'myusername',
     principal_type => xs_acl.ptype_db)
     );
    END;
    /
  select * FROM dba_network_acls;
  
  see also
  DBMS_NETWORK_ACL_ADMIN.CHECK_PRIVILEGE_ACLID
  DBMS_NETWORK_ACL_UTILITY.CONTAINS_HOST
